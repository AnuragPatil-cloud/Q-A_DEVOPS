# ☁ Microsoft Azure Interview Questions & Answers (DevOps)

This section contains commonly asked **Microsoft Azure interview questions** for DevOps engineers.

---

# 1. What is Microsoft Azure?

Microsoft Azure is a **cloud computing platform provided by Microsoft that offers services such as computing, storage, networking, and databases**.

It allows organizations to build, deploy, and manage applications through Microsoft-managed data centers.

---

# 2. What are the main services provided by Azure?

Azure provides various cloud services such as:

- Virtual Machines (VM)
- Azure Storage
- Azure Kubernetes Service (AKS)
- Azure App Services
- Azure Virtual Network
- Azure Functions
- Azure DevOps

---

# 3. What is Azure Virtual Machine?

Azure Virtual Machine (VM) is a **cloud-based virtual server that allows users to run applications and services**.

It provides scalable compute resources similar to AWS EC2.

---

# 4. What is Azure Resource Group?

A Resource Group is a **logical container that holds related Azure resources** such as VMs, storage accounts, and networks.

It helps manage and organize resources.

---

# 5. What is Azure Virtual Network (VNet)?

Azure Virtual Network is a **private network in the Azure cloud used to securely connect Azure resources**.

It allows communication between:

- virtual machines
- databases
- applications

---

# 6. What is Azure Storage?

Azure Storage is a **cloud storage service used to store large amounts of data**.

Types of storage:

- Blob Storage
- File Storage
- Queue Storage
- Table Storage

---

# 7. What is Azure Blob Storage?

Blob Storage is used for **storing large unstructured data such as images, videos, logs, and backups**.

It is similar to **Amazon S3**.

---

# 8. What is Azure Kubernetes Service (AKS)?

AKS is a **managed Kubernetes service that simplifies deploying and managing containerized applications**.

Azure manages the Kubernetes control plane.

---

# 9. What is Azure App Service?

Azure App Service is a **platform-as-a-service (PaaS) used to build and host web applications, APIs, and mobile backends**.

It supports multiple programming languages.

---

# 10. What is Azure DevOps?

Azure DevOps is a **set of development tools used for CI/CD pipelines, source control, and project management**.

Key services include:

- Azure Repos
- Azure Pipelines
- Azure Boards
- Azure Artifacts

---

# 11. What is Azure Functions?

Azure Functions is a **serverless compute service that runs code in response to events**.

It automatically scales based on demand.

---

# 12. What is Azure Load Balancer?

Azure Load Balancer distributes **incoming traffic across multiple virtual machines** to ensure high availability.

---

# 13. What is Azure Monitor?

Azure Monitor is a **monitoring service used to collect metrics, logs, and performance data from Azure resources**.

It helps detect and troubleshoot issues.

---

# 14. What is Azure Key Vault?

Azure Key Vault is used to **securely store sensitive information such as secrets, passwords, API keys, and certificates**.

---

# 15. What is Azure Active Directory (Azure AD)?

Azure Active Directory is a **cloud-based identity and access management service** used for authentication and authorization.

---

# 16. What is Azure Availability Zone?

Availability Zones are **physically separate data centers within an Azure region** used to ensure high availability and fault tolerance.

---

# 17. What is Azure Autoscaling?

Azure Autoscaling automatically **adjusts resources based on traffic or workload demand**.

This improves performance and cost efficiency.

---

# 18. What is Infrastructure as Code in Azure?

Infrastructure as Code (IaC) in Azure allows provisioning infrastructure using code.

Common tools include:

- Terraform
- Azure ARM templates
- Bicep

---

# 19. What is the difference between Azure VM and Azure App Service?

| Feature | Azure VM | Azure App Service |
|---|---|---|
Type | Infrastructure as a Service | Platform as a Service |
Control | Full OS control | Managed platform |
Use Case | Custom server workloads | Web applications |

---

# 20. Why is Azure important for DevOps?

Azure helps DevOps teams by providing:

- scalable cloud infrastructure
- CI/CD pipelines
- container orchestration
- monitoring tools
- automation capabilities
